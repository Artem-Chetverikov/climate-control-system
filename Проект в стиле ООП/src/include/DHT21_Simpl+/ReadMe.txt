DHT21_Simpl

Библиотека взаимодействия с датчиком температуры и влажности DHT21


Пример:
#include "DHT21_Simpl.h"


DHT21_Simpl_Temp	DHT_1_temp(10);			//определяем датчик на пине №10 для запроса и хранения температуры

DHT21_Simpl_Hum		DHT_1_hum( DHT_1_temp);		//определяем экземпляр для хранения влажности

DHT_1_temp.Read();	//опрос датчика
DHT_1_hum.Read();	//запрос влажности 


t = DHT_1_temp.ReturnVal();				
h = DHT_1_hum.ReturnVal();				
			

err1 = DHT_1_temp.ReturnErr();		//вывод количества ошибок при опросе
err1 = DHT_1_hum.ReturnErr();
